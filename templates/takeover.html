Request WG to take over your plugin or project

There are some criterias to respect before we can accept the project:

  1. Usefulness
    • The project must be a useful one
  2. Documentation
    • The project must documented upto a nice level
  3. Tests
    • Tests must have a high coverage and be easy to run
Then mail to flaskcwg at python.org with the project detail as well as why you want to relinquish the project.
